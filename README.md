# vscode-settings
Manage VScode settings by GitHub.

## Supported OS
MacOS, Linux

## Getting started
```
git clone git@github.com:otashun/vscode-settings.git
cd vscode-settings
./install.sh
```

## Usage
### install
```
cd vscode-settings
./install.sh
```

### reflect local extensions
```
cd vscode-settings
./reflect_local_extensions.sh
```
